# SolutionBase - Custom Website Development

A modern, professional web design portfolio showcasing custom-built websites for entrepreneurs and businesses in the Netherlands.

## About

SolutionBase specializes in creating tailored websites that strengthen your online presence. We focus on modern design, clean code, and short communication lines to deliver websites that actually work for your business.

## Features

- **Responsive Design** - Mobile-first approach with modern CSS and Tailwind
- **Fast Performance** - Optimized for speed and search engines
- **Custom Solutions** - Bespoke websites built to your exact needs
- **Professional Portfolio** - Showcasing completed projects from photographers, chefs, and more

## Project Process

1. **Bespreken** (Discuss) - Understand your vision and requirements
2. **Opzet** (Setup) - Create an initial version for your feedback
3. **Bouwen** (Build) - Develop and refine the website
4. **Online** (Launch) - Deploy and make your site live

## Pages

- **Home** (`index.html`) - Main landing page with services overview
- **Diensten** (`diensten.html`) - Detailed service offerings
- **Over mij** (`over-mij.html`) - Company information and background
- **Contact** (`contact.html`) - Get in touch

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom stylesheets with Tailwind CSS
- **JavaScript** - Interactive components and navigation
- **Tailwind CSS** - Utility-first CSS framework
- **Google Fonts** - Kanit font family

## Deployment

Hosted on **Vercel** with clean URLs enabled via `vercel.json` configuration.
